# Peculiar Velocities using SNe

Repository for running MCMC chains using Peculiar Velocities
See [Crisman et al. (2026)](https://arxiv.org/abs/2604.06361)

## Structure

- `emcee/` - scripts to run likelihoods in emcee
- `cobaya_likelihoods/` - likelihoods for cobaya (should be placed in your/path/to/cobaya/likelihoods/)
- `cobaya_yamls/` - .yaml files for cobaya
- `sne_functions/` - functions used for running chains with emcee
